# 🧑‍💼 Proyecto Integrador - Sistema de Gestión de Personal

Aplicación de consola en Java orientada a la **gestión de empleados de distintos sectores**. El proyecto fue desarrollado aplicando **Programación Orientada a Objetos (POO)**, principios **SOLID**, control de versiones con **Git**, y buenas prácticas de diseño.

---

## 🎯 Objetivo

Desarrollar una aplicación en consola que permita:

- Gestionar empleados de diferentes tipos (Programadores, Diseñadores, Técnicos, Administrativos)
- Aplicar conceptos avanzados de Java: herencia, clases abstractas, interfaces, polimorfismo
- Integrar principios SOLID y patrones de diseño básico
- Demostrar habilidades de estructuración, modularidad y control de versiones

---

## ⚙️ Funcionalidades actuales

- ✅ Alta, baja y modificación de empleados (por legajo)
- ✅ Visualización completa de empleados con tareas específicas
- ✅ Clases abstractas y herencia para modelar tipos de empleados
- ✅ Interface `Imprimible` para informes personalizados
- ✅ Separación de responsabilidades por clase y método
- ✅ Control básico de errores en entradas de usuario
- ✅ Uso de colecciones dinámicas (ArrayList)

---

## 🧩 Tecnologías y formato

- **Lenguaje:** Java 8+
- **Entrada/Salida:** Consola con Scanner
- **Control de versiones:** Git y GitHub
- **Organización:** Clases distribuidas por funcionalidad con comentarios explicativos
- **IDE sugerido:** IntelliJ IDEA / Eclipse / NetBeans
